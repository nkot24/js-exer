1. Uzdevums: Izveido "To-Do" saraksta aplikāciju

Šajā uzdevumā izveidosi vienkāršu "To-Do" sarakstu, kurā var pievienot jaunus uzdevumus.

  1. HTML failā izveido <input> elementu un <button> ar tekstu "Pievienot uzdevumu".
  2. Izveido <ul> elementu, kur tiks pievienoti uzdevumi kā saraksta elementi.
  3. JavaScript failā:
    • Pievieno click event listener button elementam.
    • Event listener funkcijā izveido jaunu <li> elementu ar uzdevuma tekstu no input lauka un pievieno to <ul> sarakstam.
    • Pēc uzdevuma pievienošanas notīri input lauku.
  4. Pārbaudi, vai vari pievienot jaunus uzdevumus sarakstam, izmantojot pogu. 
// Izvēlies elementus no HTML
        const button = document. ...;
        const input = document. ...;
        const ul = document. ...;
        
        // Pievieno click event listener
        button.addEventListener('click', function() {
            // Izveido jaunu li elementu
            const li = document.createElement('li');
            li.textContent = ...;
        
            // Pievieno jauno li elementu ul sarakstam
            ul.appendChild(...);
        
            // Notīra input lauku
            ...
        });

2. Uzdevums: Pievieno pogu uzdevumu dzēšanai

Šajā uzdevumā uzlabosi "To-Do" sarakstu, pievienojot pogu katram uzdevumam, lai to varētu dzēst.

  1. Katrā pievienotajā uzdevuma elementā (<li>) izveido pogu ar tekstu "Dzēst".
  2. Pievieno click event listener katrai "Dzēst" pogai, lai, nospiežot to, attiecīgais uzdevums tiktu noņemts no saraksta.
  3. Pārbaudi, vai vari pievienot un dzēst uzdevumus sarakstā.
// Izvēlies HTML elementus
    
    // Pievieno event listener pievienošanas pogai
    button.addEventListener('click', function() {
        ...
    
        // Izveido dzēšanas pogu un pievieno to li elementam
        const deleteButton = document.createElement('button');
        deleteButton.textContent = 'Dzēst';
        li.appendChild(...);
    
        // Pievieno dzēšanas funkcionalitāti
        deleteButton.addEventListener('click', function() {
            ...
        });
    
        // Pievieno li elementu ul sarakstam un notīra ievades lauku
        ul.appendChild(...);
        ...
    });
