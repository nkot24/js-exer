1. Jaunu elementu pievienošana DOM

JavaScript ļauj mums dinamiski pievienot jaunus elementus DOM. To var panākt ar tādām metodēm kā appendChild, insertAdjacentHTML un citām.

2. Izmantojot document.createElement() un appendChild()

Šī ir viena no visbiežāk izmantotajām metodēm. Vispirms izveido jaunu elementu un pēc tam pievieno to kādam vecākelementam. 

// 1. solis: izveido jaunu elementu
const newElement = document.createElement('div');

// 2. solis: iestata saturu vai atribūtus
newElement.textContent = 'Sveika, pasaule!';
newElement.className = 'new-class';

// 3. solis: pievieno DOM
document.body.appendChild(newElement);

3. Izmantojot insertAdjacentHTML()

insertAdjacentHTML() ļauj mums ievietot HTML kodu dažādās vietās attiecībā pret elementu. 

// Ievieto HTML pirms elementa beigām
const parentElement = document.getElementById('container');
parentElement.insertAdjacentHTML('beforeend', '
Jauns Elements
');

4. Izmantojot innerHTML

innerHTML ļauj mums tieši ievietot HTML, taču tas aizvieto visu esošo saturu. 

// Brīdinājums: innerHTML aizvieto esošo saturu
const container = document.getElementById('container');
container.innerHTML += '
Pievienots ar innerHTML
';

5. Izmantojot prepend() un append()

prepend() pievieno elementus vecākelementa sākumā, kamēr append() pievieno tos beigās. 

// Pievieno sākumā
parentElement.prepend(newElement);

// Pievieno beigās
parentElement.append(newElement);

6. Secinājums

Šīs ir dažas no galvenajām metodēm, kā pievienot elementus DOM, izmantojot JavaScript.

Katrai no tām ir savi pielietošanas gadījumi un priekšrocības, atkarībā no projekta prasībām.

1. Uzdevums: Izveido un pievieno elementu

  1. Izmantojot document.createElement(), izveido <p> elementu ar tekstu "Jauns paragrāfs".
  2. Pievieno šo elementu body elementam, izmantojot appendChild().
  3. Pārbaudi, vai paragrāfs parādās lapā.

2. Uzdevums: Ievieto HTML ar insertAdjacentHTML

  1. Izveido <div> elementu ar id "container" HTML kodā.
  2. Izmanto insertAdjacentHTML(), lai ievietotu <p>Vēl viens paragrāfs</p> tieši pirms container elementa beigām.
  3. Pārliecinies, vai jaunais paragrāfs parādās iekš container elementa.

3. Uzdevums: Pievieno elementu sākumā

  1. Izveido <ul> ar dažiem <li> elementiem HTML kodā.
  2. Izmanto prepend(), lai pievienotu <li>Jauns elements sākumā</li> <ul> elementa sākumā.
  3. Pārbaudi, vai jaunais li elements parādās saraksta sākumā.

4. Uzdevums: Pievieno elementu ar innerHTML

  1. Izveido <div> elementu ar id "content" HTML failā.
  2. Izmanto innerHTML, lai pievienotu <h2>Virsraksts</h2> un <p>Šis ir teksts</p> elementu iekš "content".
  3. Pārbaudi, vai abi elementi tiek pievienoti pareizi.
